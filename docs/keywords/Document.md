
# Document

## SYNOPSIS
Defines a named block that can be called to output documentation.

## SYNTAX

```
Document [-Name] <String>
```

## EXAMPLES

### EXAMPLE 1

```powershell
Document 'Test' {
    # Insert content here
}
```

## PARAMETERS

### -Name
The name of the document.
